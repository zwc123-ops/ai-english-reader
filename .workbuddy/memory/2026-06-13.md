## 项目概览

查看了 `D:\English reading` 项目全貌：

- **项目名**：AI 英语精读阅读器（个人桌面版原型，v0.1）
- **运行方式**：`npm start` → `node server.js`，访问 http://127.0.0.1:18789/
- **技术栈**：Node.js 原生 HTTP 服务（无框架） + 原生 JS 前端（无构建工具）
- **文件结构**：server.js（后端）+ app.js（前端）+ index.html + styles.css
- **AI 接入**：DeepSeek，通过 `/api/deepseek` 本地代理转发，支持 HTTP 代理和直连双模式，带重试
- **词典**：`/api/dictionary` 代理 dictionaryapi.dev，内置约 80 个常用词种子词典
- **存储**：IndexedDB（书籍、进度）+ 浏览器存储（设置）
- **已实现功能**：书库、TXT/EPUB/PDF 阅读、章节切分、阅读进度、字号/主题、选词查词、生词收藏与复习、高亮与笔记、导出 Markdown/CSV、AI 查词/句子分析/段落解释
- **规划文档**：DEV_DOCUMENT.md（完整需求和里程碑）+ AI_READER_INTERNAL_API.md（接口契约）
